📖 Como Usar

Formatos Disponíveis:

  • SVG: Vetor escalável, sem perda de qualidade, ideal para web
  • PNG: Imagem bitmap em 3 resoluções (1x, 2x, 4x) para Canva e redes sociais
  • Resolução 4x: Até 1600x480px (logos) e 800x800px (ícone) - perfeito para impressão

📐 Qual resolução escolher?

  • 1x (Padrão): Para web, perfis de redes sociais
  • 2x (Alta): Para Retina displays, posts em HD
  • 4x (Máxima): Para Canva, impressão, banners, thumbnails do YouTube

Especificações:

  • Fonte: Fira Code (monospace, tech style)
  • Cores principais: #00d4ff (azul) e #00ff88 (verde)
  • Gradiente: Linear de azul para verde
  • Efeito: Glow/sombra para destaque

🎨 Usar no Canva:

  • Clique em "Download PNG" e escolha resolução Máxima (4x)
  • No Canva: Upload → Faça upload de arquivos
  • Arraste o PNG para seu design
  • Use a versão transparente para sobrepor em qualquer fundo
  • Use a versão com background para designs minimalistas

Onde usar:

  • Header/Navbar do site
  • Footer
  • Redes sociais (perfil, cover, stories)
  • Posts no Instagram, LinkedIn, Facebook
  • Thumbnails do YouTube
  • Assinatura de email
  • Apresentações e documentos
  • GitHub profile
  • LinkedIn banner

Variações de cor:

Os arquivos SVG podem ser editados em qualquer editor de texto ou software de design (Figma, Illustrator, Inkscape) para alterar cores conforme necessário.
